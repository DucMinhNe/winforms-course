# Exercise — Lesson 08

1. Add an "I agree" `CheckBox`. Bind a "Continue" `Button`'s `Enabled` to the
   checkbox via `CheckedChanged`.
2. Put three RadioButtons (Small/Medium/Large) inside a `GroupBox`. Show the
   selection in a label that updates on `CheckedChanged`.
3. Add a **second** `GroupBox` with Color radios (Red/Green/Blue). Confirm that
   picking a color doesn't reset the size group — separate containers = separate
   groups.

**Done when:** Continue is disabled until agreement, the size label tracks the
selected radio, and the two radio groups are independent.
