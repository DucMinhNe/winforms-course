# Exercise — Lesson 11

1. In a `for` loop, create 5 buttons. Set each one's `Tag` to its number and
   `Name` to `btnN`.
2. Attach one shared `Click` handler that reads `((Button)sender).Tag` and shows
   it in a label.
3. Add a "Toggle" button that flips a textbox's `Enabled` (and try `Visible`).
4. Set `TabIndex` on the inputs so pressing **Tab** moves through them in a
   sensible order.

**Done when:** five generated buttons share one handler via `Tag`, the toggle
enables/hides a control, and tab order is intentional.
