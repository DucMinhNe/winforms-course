# Exercise — Lesson 06

1. Build the greeter: `Label` + `TextBox` + `Button`, with an output `Label`.
2. On **Greet**, show `Hello, <name>!`; if empty, show "Please enter a name."
3. Set the form's `AcceptButton` so Enter works too.
4. Add a second `TextBox` for a password with `UseSystemPasswordChar = true`;
   on click, append `(<n> char password)` to the output.

**Done when:** the greeter responds to both the button and Enter, handles empty
input, and masks the password field.
