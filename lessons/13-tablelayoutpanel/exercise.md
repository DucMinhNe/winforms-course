# Exercise — Lesson 13

1. Create a `TableLayoutPanel` docked `Fill` with 2 columns: column 0 Absolute
   90px (labels), column 1 Percent 100 (inputs).
2. Rows for **Name**, **Email**, **Message**. Make the Message row
   `SizeType.Percent` and put a `Multiline` textbox there so it grows.
3. Add a **Send** button in the last row, right-anchored.
4. Resize the window — inputs widen, the message box grows, labels stay fixed.

**Done when:** the form is fully resizable with zero hard-coded control
positions.
