# Exercise — Lesson 12

1. Add a search `TextBox` anchored `Top | Left | Right` so it widens as the form
   grows.
2. Add OK/Cancel buttons anchored `Bottom | Right` so they stick to the corner.
3. Add a `ListBox` (or panel) that fills the remaining middle area
   (`Anchor = Top|Bottom|Left|Right`, or `Dock = Fill` inside a panel).
4. Set a `MinimumSize` and resize the window — confirm nothing overlaps or
   clips.

**Done when:** resizing widens the search box, keeps the buttons in the corner,
and grows the content region.
