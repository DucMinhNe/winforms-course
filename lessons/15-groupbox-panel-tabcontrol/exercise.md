# Exercise — Lesson 15

1. Add a `TabControl` docked `Fill` with two `TabPage`s: "General" and
   "Account".
2. On **General**, add a `GroupBox` "Appearance" with two checkboxes.
3. On **Account**, add a label + textbox for a display name (notice the
   coordinates are relative to the tab page, not the form).
4. Add a bottom status label that shows the active tab via
   `SelectedIndexChanged`.

**Done when:** switching tabs updates the status label, and the GroupBox keeps
its options visually grouped.
