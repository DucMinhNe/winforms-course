# Exercise — Lesson 05

1. Build a `DialogForm` that is:
   - fixed size (`FormBorderStyle.FixedDialog`, `MaximizeBox = false`),
   - centred on its parent (`CenterParent`),
   - hidden from the taskbar (`ShowInTaskbar = false`).
2. Add Save/Cancel buttons and set `AcceptButton`/`CancelButton` so **Enter**
   saves and **Esc** cancels.
3. Experiment: set `TopMost = true`, then `Opacity = 0.85`, and re-run to see
   each effect.

**Done when:** the dialog can't be resized or maximised, Enter/Esc work, and you
can describe what `AcceptButton` does.
